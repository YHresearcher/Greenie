# The Sovereign Harvest - Landing Page

Website Landing Page chuyên nghiệp cho lĩnh vực xuất khẩu nông sản và OEM.

## Công nghệ sử dụng
- **Tailwind CSS**: Framework thiết kế nhanh.
- **AOS**: Hiệu ứng cuộn trang mượt mà.
- **EmailJS**: Tích hợp hệ thống gửi email trực tiếp từ trình duyệt.

## Hướng dẫn cấu hình Email
Để nhận được mail từ Form liên hệ, bạn cần:
1. Đăng ký tại [EmailJS](https://www.emailjs.com/).
2. Tạo **Service** và **Template**.
3. Mở file `js/main.js` và thay thế các mã:
   - `YOUR_PUBLIC_KEY`
   - `YOUR_SERVICE_ID`
   - `YOUR_TEMPLATE_ID`

## Tri khai trên GitHub Pages
1. Push code lên repository của bạn.
2. Vào phần **Settings > Pages**.
3. Chọn nhánh `main` và thư mục `/ (root)` để kích hoạt website.
